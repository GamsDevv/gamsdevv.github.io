What Gams Task Can Do

A full-featured macro suite with everything you need to automate repetitive workflows.

⏺️

Task Recording

Record any sequence of actions and replay them on demand. Named tasks keep your macros organized.

🔤

Wait OCR

Pause execution until specific text appears on screen using optical character recognition before continuing playback.

📐

Wait Area

Define screen regions and wait for changes or activity within a specific area before advancing.

🎨

Wait Color

Detect pixel colors and hold execution until the target color appears — perfect for game automation.

📋

Step Management

Insert, edit, and remove individual steps from any task with full control over execution order.

⚡

Hotkey Control

Full keyboard shortcut support — record, play, stop, and pick without touching the mouse.

How To Use

Up and running in under two minutes.

SETUP_GUIDE.md
1

Join Discord & Download

Click the Download button above. You'll be redirected to our Discord server where you can grab the latest .ahk file.

2

Install AutoHotkey

Download and install AutoHotkey v1.1 from autohotkey.com if you haven't already.

3

Run the Script

Double-click GamsTask.ahk to launch the macro panel. It will appear in your system tray.

4

Create a Task

Press F1 or click Record to start recording. Perform your actions, then press F9 to stop.

5

Play It Back

Select your saved task and press F8 to replay it any time you need it.
